    Awesome Koa

    Github

    Project

    • strapi/strapi: 🚀 Open source Node.js Headless CMS to easily build customisable APIs

    Framework

    • xiongwilee/Gracejs: A Nodejs BFF framework, build with koa2（基于koa2的标准前后端分离框架）

    • nobitajs/nobita: Nobita 是一个基于Koa而诞生的一款框架。

    Middleware

    • jeffijoe/awilix-koa: Awilix helpers/middleware for Koa 2
    • kudos/koa-websocket: Light wrapper around Koa providing a websocket middleware handler that is koa-route compatible.
    • rkusa/koa-passport: Passport middleware for Koa
    • shellscape/koa-webpack: Development and Hot Reload Middleware for Koa2
    • m4nuC/async-busboy: Promise based multipart form parser for KoaJS
    • Secbone/koa-session2: Middleware for Koa2 to get/set session
    • JasonBoy/koa-web-kit: 🚀A Modern, Production-Ready, and Full-Stack Node Web Framework with React
    • koajs/userauth: koa user auth middleware
    • vagusX/koa-proxies: a koa@2.x proxy middleware
    • longztian/koa-session-minimal: Minimal implementation of session middleware for Koa 2
    • dominhhai/koa-log4js: log4js-node supports Koa-middleware
    • koajs/mock: Simple web page mock middleware
    • ishen7/koa2-connect-history-api-fallback: Fallback to index.html for applications that are using the HTML 5 history API
    • dominhhai/koa-busboy: Upload files with Busboy
    • imingyu/koa-vue-view: A Koa view engine which renders Vue components on server.
    • keenwon/subdomain: Simple and lightweight Koa middleware to handle multilevel and wildcard subdomains
    • tunnckoCore/koa-ip-filter: koa middleware to filter request IPs or custom ID with glob patterns, array, string, regexp or matcher function. Support custom 403 Forbidden message and custom ID.
    • JefferyLiang/koa-2-acl: ACL middleware of koa 2

    Tools

    • koajs/joi-router: Configurable, input and output validated routing for koa
    • graphql-community/koa-graphql: Create a GraphQL HTTP server with Koa.
    • janvotava/swagger-koa: Swagger + Koa = {swagger-koa}

    Boilerplate

    • jeffijoe/koa-es7-boilerplate: A boilerplate for writing Koa 2 apps in ES7 with Babel.
    • i5ting/koa-generator: Koa' application generator for 1.x and 2.x（ Express-style and support all middlewares include async/await ）
    • yi-ge/Koa2-API-Scaffold: 一个基于Koa2的轻量级RESTful API Server脚手架。
    • javieraviles/node-typescript-koa-rest: REST API boilerplate using NodeJS and KOA2, typescript. Logging and JWT as middlewares. TypeORM with class-validator, SQL CRUD. Docker included. Swagger docs, actions CI and valuable README
    • chikara-chan/react-isomorphic-boilerplate: 🌟 An universal React isomorphic boilerplate for building server-side render web app.
    • doodooke/doodoo.js: Doodoo.js -- 中文最佳实践Node.js Web快速开发框架，支持Koa.js, Express.js中间件。
    • Talento90/typescript-node: 📋 TypeScript and NodeJS project template.
    • clarkdo/hare: 🐇 Application boilerplate based on Vue.js 2.x, Koa 2.x, Element-UI and Nuxt.js
    • adrianObel/koa2-api-boilerplate: Boilerplate for building APIs with koa2.
